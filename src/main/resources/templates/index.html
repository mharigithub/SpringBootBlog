Automation Testing Blog

This blog provides short and straight to the point solutions to most of the Automation Testing concepts that are in demand with current Industry leading companies in the market.

There are many technologies and frameworks to automate tests, here we focus on the most popular and widely used framework i.e. Selenium and Appium with Java programming language.

Selenium combined with java is widely accepted combination by many Organizations. Another advantage of using Selenium with Java is, we get huge community support from internet who guides and also provide solutions to the problems that you encounter during automating the tests.

This blog provides solutions to Appium concepts too. Appium is a widely used mobile automation testing framework. It can automate both iOS and Android devices. Similar to Selenium, Appium combined with Java also has a huge community support in the internet.

Other topics covered by this blog include TestNG, Cucumber, Jenkins, Git, Jira, Page Object Model, Maven, Parallel Testing etc.
