Xpath

Different xpath techniques

1) Using dynamic attribute values: 

String str="button";
driver.findElement(By.xpath("//*[@label="+str+"]")));

2) For example, if there are two buttons with label USD and to click on the USD button which is next to Amount field, xpath can be written as below. Using this xpath we can click on USD button 

//*[@text='Amount']/..//*[starts-with(text(),'USD')]

3) Using or in xpath: 

//*[contains(@name,"xyz") or contains(@label,"abc")]
